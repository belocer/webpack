##### Установить npm зависимости
```
npm install
```
##### Разработка

Для запуска веб-сервера с лайврелоадом
```
npm start
```
Для создания билда
```
npm run build
```
Для генерации сетки Smart-grid Дмитрия Лаврика
```
npm run build
```
